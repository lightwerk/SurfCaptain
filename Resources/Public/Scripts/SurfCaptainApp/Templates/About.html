SurfCaptain (v)

your friend and helper.

Project on Bitbucket.

Features

  • Deploy any TYPO3 project from your git to a destination.
  • Sync your development system with the production database and files.
  • Get every project member a copy of the project in no time.
  • Put the customers server configurations under version control.
  • See which commit is currently live on which server.
  • Keep a deployment history.
